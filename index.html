Introducción a CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento HTML. Permite definir cómo se verán los elementos en la página web, incluyendo colores, fuentes, márgenes, y más.

2. Funcionamiento Básico de CSS

CSS funciona aplicando estilos a elementos HTML mediante selectores. Cuando se carga una página web, el navegador procesa el HTML y aplica los estilos CSS para renderizar el contenido visualmente. Los estilos pueden ser específicos (aplicados a un solo elemento) o globales (aplicados a múltiples elementos).

3. Incluir CSS en un archivo HTML

Hay tres maneras principales de incluir CSS en un archivo HTML:

  • Inline CSS: Usando el atributo style dentro de una etiqueta HTML. 
    <p style="color: red;">Este es un texto rojo.</p>
  • Internal CSS: Usando una etiqueta <style> dentro de <head>. 
    <head>
    <style>
        p { color: blue; }
    </style>
</head>
  • External CSS: Vinculando un archivo CSS externo con la etiqueta <link>. 
    <head>
    <link rel="stylesheet" href="styles.css">
</head>

4. Definición de una regla CSS

Una regla CSS es una declaración que define cómo deben ser los estilos aplicados a ciertos elementos. Se compone de un selector y un bloque de declaración.

5. Elemento de una regla CSS

  • Regla: Conjunto de instrucciones CSS.
  • Selector: Parte de la regla que especifica a qué elemento(s) se aplican los estilos. Ejemplo: h1, .clase, #id.
  • Bloque de declaración: Contiene uno o más pares de propiedad-valor, encerrados en llaves {}.
  • Propiedad: Característica del elemento que se desea modificar (por ejemplo, color, font-size).
  • Valor: Especifica el valor que se asignará a la propiedad (por ejemplo, red, 16px).

Ejemplo de una regla CSS:

h1 {
    color: red; /* propiedad: valor */
    font-size: 24px;
}

6. Medios CSS

Los medios CSS permiten aplicar estilos específicos según el tipo de dispositivo o tamaño de pantalla. Se utilizan con la regla @media. Por ejemplo:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

7. Comentarios CSS

Los comentarios en CSS se utilizan para agregar notas o descripciones en el código. Se escriben entre /* y */ y no afectan la ejecución del CSS.

/* Este es un comentario */
h1 {
    color: blue; /* Este texto será azul */
}

8. Sintaxis de la definición de las propiedades CSS

La sintaxis para definir propiedades CSS es la siguiente:

selector {
    propiedad: valor; /* cada declaración termina con un punto y coma */
}

Ejemplo:

p {
    font-size: 16px; /* Tamaño de fuente */
    color: black; /* Color del texto */
}
